- `cout` -> In js is pre-defined object only which is used for printing something inside the output window and it is defined inside the iostream header file. Anything written inside the double quoted *""* gets printed as it is in the output screen.
- */n* -> is used for printing to giving a one line break or we an say one line space.
- *endl* -> even we can write endl to provide a line break at the end of the cout statement.
- But `/n` is preffered more than the `endl` because '/n' has fast processing than the 'endl' and this is because the endl flushes the output stream.
- The whole of our cpp program or code runs in a sequential manner.
